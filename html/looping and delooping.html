looping and delooping

Last modified: 2025-12-11

In $(\infty,1)$-categories

Let $C$ be an (∞,1)-category with a terminal object $*$.

For any pointed object $X$ in $C$, its loop space object (AKA looping) of $X$, denoted $\Omega X$, is defined as the (∞,1)-pullback of the point $x_{0}$ along itself in $C$:

A delooping of an object $A\in C$ is a pointed object $\mathrm{B}A$ such that there is an equivalence $A \simeq \Omega(\mathrm{B}A)$ in $C$.
