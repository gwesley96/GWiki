Idea. Categories abstract the common structure shared by mathematical contexts: objects (like sets, groups, topological spaces) and structure-preserving maps between them (functions, homomorphisms, continuous maps), formalized through composition and identities.

test

Definition (Category). A category $\cC$ consists of:

  • A class $\Obj(\cC)$ of \emph{objects}
  • For each pair $A, B$, a set $\Hom(A, B)$ of \emph{morphisms}
  • An identity $\id_A \colon A \to A$ for each object
  • Composition $\circ \colon \Hom(B,C) \times \Hom(A,B) \to \Hom(A,C)$

satisfying associativity and unit laws.

Examples

Example. Standard categories:

  • $\Set$: sets and functions
  • $\Grp$: groups and homomorphisms
  • $\Top$: topological spaces and continuous maps
  • $\Vec_k$: vector spaces and linear maps
