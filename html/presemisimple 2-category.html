presemisimple 2-category

Last modified: December 12, 2025 at 10:15 PM ET
Created: June 21, 2025 at 12:51 PM ET

Definition 2.26. A linear 2-category $\mathfrak{C}$ is presemisimple if all $n$-fold linking algebras are semisimple multitensor categories. A presemisimple 2-category is finite if all $n$-fold linking algebras are multifusion and there is a global bound on the dimensions of $\mathrm{End}(1)$ for the centers of all linking algebras. (That is, there is a $K>0$ such that for any linking algebra $\mathcal{L}=\mathcal{L}(a_1,\dots, a_n)$, $\dim(\mathrm{End}(1_{\mathcal{Z}(\mathcal{L})}))<K$.)

If $a,b\in\mathfrak{X}$ are in distinct components of a presemisimple 2-category $\mathfrak{X}$, then the multitensor linking category ($\mathrm{E}_{1}$-algebra) $\mathcal{L}(a,b)$ decomposes as $\mathcal{L}(a)\boxplus \mathcal{L}(b)$ (direct sum of categories), and so its Drinfeld center splits as $\mathcal{Z}(\mathcal{L}(a,b))\cong \mathcal{Z}(\mathcal{L}(a))\oplus \mathcal{Z}(\mathcal{L}(b))$. Thus, when $\mathfrak{X}$ is finite presemisimple, $\mathcal{Z}(\mathfrak{X})$ splits as a direct sum $\boxplus_s$ of indecomposable braided fusion categories, and these summands correspond to the indecomposable fusion blocks of $\mathcal{L}(a,b)$.

Definition 1.2.7 (Alternative to the above). A presemisimple 2-category is a rigid, locally semisimple 2 category wherein each object decomposes into a finite direct sum of simple objects and with the property that any (hence all) of the following equivalent conditions are satisfied:

  • (pSS1) endomorphism categories of simple objects are multitensor;
  • (pSS2) (Proposition 1.2.19) composition of nonzero morphisms between simple objects is nonzero; and
  • (pSS3) (Corollary 1.2.15) every object decomposes into a finite direct sum of objects with simple identity 2-morphism.

Definition 1.2.8. A presemisimple 2-category is finite if it is locally finite semisimple and if it has finitely many equivalence classes of simple objects.

Properties

Remark 1.2.10. Endomorphism categories in a finite presemisimple 2-category are multifusion.

Example 1.2.11. Every infusion category $\mathcal{C}$ deloops to a presemisimple 2-category $\mathrm{B}\mathcal{C}$.

Corollary 1.2.16. Projections and inclusions of 2-direct sums of simple objects are simple.

Proposition 1.2.17 (Simple if and only if indecomposable). An object in a presemisimple 2-category is simple if and only if it is indecomposable.

Proof. Any nontrivial decomposition $x=x_{1}\boxplus x_2$ gives an injective (fully faithful) inclusion 1-morphism $x_1\to x$, and so $x_{1}$ is a nontrivial subobject, meaning $x$ is not simple. Conversely, by presemisimplicity, any object is a sum of simple objects, so an indecomposable object must have exactly one summand, and so is itself simple.

Proposition 1.2.21. Decomposition into simples is unique (contractible choice) in presemisimple 2-categories.
