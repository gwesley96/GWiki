\Title{3-Manifold}
\Tags{low-dimensional-topology, 3-manifolds}
\Summary{Orientable 3-manifolds can be built by Dehn surgery on framed links in $S^3$, so surgery data presents every example.}

\begin{document}

\KeySources{\nlab{3-manifold}, \wref{Dehn surgery}}

\NoteHeader

\begin{framedidea}
Low-dimensional topology packages 3-manifolds via surgery: starting from $S^3$, perform Dehn surgery along framed links to obtain every orientable 3-manifold.
\end{framedidea}

By \wref{Dehn surgery}, every \wref{orientable manifold} 3-manifold can be presented as a pair $(K,r)$ where $K$ is a \wref{knot} in $S^{3}$ and $r=p/q\in\mathbb{Q}\cup\{\infty\}$ (called the \textbf{surgery coefficient}).
